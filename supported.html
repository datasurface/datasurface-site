Understanding the Columns

Primary Storage

The source of truth for ingested data. This is where DataSurface writes data after ingestion and transformation. Platforms that support primary storage can hold your master datasets.

Compute Engine

Executes data transformations and pipeline logic. Platforms like AWS (via EKS/Lambda), Snowflake, and Databricks provide compute capabilities. Traditional databases rely on external compute via Kubernetes.

CQRS Storage

Optimized replicas for consumer queries. DataSurface replicates data to Consumer Replica Groups (CRGs) tailored to specific workloads. Any database that can hold query-optimized views qualifies.
